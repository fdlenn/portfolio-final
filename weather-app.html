Weather App

A project from SheCodes, developing a real-time weather app using API.

Technologies I worked with:

Weather App Wireframe

Every great idea starts with a wireframe. I designed my weather app wireframe with InVision and it was based in Google's iconic weather function: clean and simple.

Creating a wireframe is good practice and helps you gather your ideas and the functional requirements before you get started on the coding.

Vanilla Weather app

I built this vanilla weather app with advanced HTML, CSS, JS and Bootstrap with integrated API. Users can search any city for realtime weather updates.

The code for this project is open-sourced on GitHub.

React Weather App

I built the same weather app on the SheCodes React workshop, but this time using React. Again, I went with a simple and clean design for the app.

The code for this project is open-sourced on GitHub.
